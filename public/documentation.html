myFlix

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies. Endpoints:
GET requests:
localhost:8080/movies (Returns a list of movies in JSON format)
localhost:8080/movies/:title (Gets a specific movie by the Title)
localhost:8080/movies/genre/:genreName (Gets movies by the Genre)
localhost:8080/movies/directors/:directorName (Gets movies by the Director)
POST requests:
localhost:8080/users (Creates a new user and needs to be in the correct JSON format. Username, Password and Email are required)
localhost:8080//users/:Username/movies/:MovieID (Creates a new movie to the favorites list of a user.)
PUT requests:
localhost:8080/users/:Username (Updates user name)
DELETE requests:
localhost:8080/users/:id/:movieTitle (Removes a movie from favorite movie list)
localhost:8080/users/:Username/ (Removes a user)
